# hackatonURJC2026SST
II Hackaton URJC - Desafío DHL 📦🌱
Este proyecto fue desarrollado durante el II Hackaton de la Universidad Rey Juan Carlos (2026), respondiendo al reto planteado por DHL para reducir la huella de carbono en la cadena de suministro y logística.

🚀 El Proyecto: Eco-Logistics MVP
Nuestra solución combina la optimización algorítmica con la concienciación del usuario final para lograr un impacto real en las emisiones de CO2.

Características principales:
Algoritmo de Reorganización: Un motor lógico que reestructura la asignación de paquetes y rutas de DHL para maximizar la eficiencia de carga y reducir kilómetros innecesarios.

Sistema de EcoPoints: Una capa de gamificación que premia a los clientes por elegir opciones de envío sostenibles (puntos de recogida, horarios valle, etc.), canjeables por beneficios.
